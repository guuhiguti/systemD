# Doa+ (Projeto Flask)

Aplicação web para gerenciamento de doações, com diferentes tipos de usuários: **colaboradores**, **instituições** e **administradores**.  
O projeto utiliza **Flask**, **SQLAlchemy** e **PostgreSQL** para armazenamento dos dados.

---

## Funcionalidades

- Cadastro e login de usuários (colaborador, instituição, administrador)
- CRUD de recursos (itens que podem ser doados)
- Registro de doações
- Painel administrativo para gerenciar usuários e recursos
- Listagem de doações por usuário e recurso

---

## Tecnologias utilizadas

- Python 3.13+
- Flask
- Flask SQLAlchemy
- PostgreSQL
- Bootstrap 5 para frontend
- Poetry para gerenciamento de dependências
